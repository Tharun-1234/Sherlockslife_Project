## 🧠 How to Remember

### Simple Memory Rule 🎯

**Only these combinations skip ID:**
- **Politician + MLA** (both relations: Spouse, Friends)
- **Politician + IAS** (only Spouse)
- **Bureaucrat + IPS** (both relations: Spouse, Friends)
- **Bureaucrat + IAS** (only Spouse)

**Everything else = ID Required**

---

## 💡 Why These Combinations Skip ID?

Because these are **high-ranking government officials** with trusted relationships:
- **MLA** = State Legislator (Member of Legislative Assembly)
- **IAS** = Top Civil Servant (Indian Administrative Service)
- **IPS** = Police Officer (Indian Police Service)
- **Spouse** = Legally married partner
- **Friends** = Trusted colleagues of officials

These are verified **automatically in government databases**, so no ID upload needed.
